Rock Paper Scissors

Let's play! You have to return which player won! In case of a draw return Draw!.

Examples(Input1, Input2 --> Output): <br>

"scissors", "paper" --> "Player 1 won!" <br>
"scissors", "rock" --> "Player 2 won!" <br>
"paper", "paper" --> "Draw!"
